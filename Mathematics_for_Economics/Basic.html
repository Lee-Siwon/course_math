Basic Number Symbol

$$\mathbb{N}$$: natural number (i.e. $$1,2,3,4$$)

$$\mathbb{Z}$$: number (i.e. $$\pm1,\pm2,\pm3,\pm4$$)

$$\mathbb{Q}$$: rational number (i.e. $$\frac{1}{3},2,3.1,4.2$$)

$$\mathbb{P}$$: irrational number (i.e. $$\sqrt{5},log 3, \pi$$)

$$\mathbb{R}$$: natural number (i.e. $$\sqrt{5},\pi,e,2.3$$)
